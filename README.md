# 📊 Power BI Sales Performance Dashboard

![Project Cover](Images/cover.png)

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze sales performance through executive-level reporting and business intelligence. It combines DAX measures, Power Query transformations, and data visualization techniques to help monitor KPIs, regional performance, customer behavior, and delivery operations.

---

## ✨ Dashboard Pages

- 📊 Executive Overview – Executive KPIs and overall sales performance.
- 📈 Sales Analysis – Product performance, promotions, and customer insights.
- 🌍 Geographic Analysis – Regional sales comparison and geographic distribution.
- 🚚 Delivery Analysis – Delivery efficiency, shipping cost, and return analysis.
